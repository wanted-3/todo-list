# Todo-app

## 실행 커맨드

```sh
npm install && npm start
```

## 구현 내용

- Todo 추가기능
- Todo 삭제기능
- Todo 검색기능
- Todo 완료 여부에 따른 필터링
- 추가된 Todo로컬스토리지에 저장해 다음에 접속시에도 확인 가능하도록 구현
